Plant Hardiness Zone Drilldown

USDA Hardiness Zones based on absolute minimum temperature have limited value in selecting appropriate species for a location.

This analysis uses hourly logged temperature to better understand what conditions are typical for a particular region. This site will ultimately evolve into an interactive app allowing the user to query by nearest station to customize the analysis. For now, a quick examination of Zone 8B is included.
